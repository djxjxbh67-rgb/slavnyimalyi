# Как обновить информацию о команде на сайте

## Что нужно сделать

Все данные о сотрудниках хранятся в **одном файле**: `data/team.json`

Чтобы обновить информацию (добавить дни рождения, девизы, образование):

### Шаг 1
Откройте ссылку:  
👉 https://github.com/djxjxbh67-rgb/slavnyimalyi/blob/main/data/team.json

### Шаг 2
Нажмите кнопку **карандаш ✏️** (Edit this file) в правом верхнем углу файла.

### Шаг 3
Найдите нужного сотрудника и впишите данные **между кавычками**. Например:

**Было:**
```json
"birthday": "",
"motto": ""
```

**Стало:**
```json
"birthday": "25 марта",
"motto": "Каждый ребёнок — это целый мир."
```

### Шаг 4
Нажмите зелёную кнопку **«Commit changes…»** внизу страницы.  
Сайт обновится автоматически через 1–2 минуты.

## 📸 Как добавить фотографии сотрудников

### Шаг 1
Подготовьте фото: квадратные, размером примерно 300×300 пикселей.  
Назовите файлы по-английски, без пробелов. Например: `yulia.jpg`, `svetlana.jpg`.

### Шаг 2
Откройте папку с фото на GitHub:  
👉 https://github.com/djxjxbh67-rgb/slavnyimalyi/tree/main/public/images/team

### Шаг 3
Нажмите **«Add file»** → **«Upload files»** и загрузите все фотографии.  
Нажмите зелёную кнопку **«Commit changes»**.

### Шаг 4
Теперь откройте файл `data/team.json` (Шаг 1 из раздела выше) и впишите имя файла:

**Было:**
```json
"photo": ""
```

**Стало:**
```json
"photo": "yulia.jpg"
```

Сохраните (Commit changes). Через 1–2 минуты на сайте появится фото вместо иконки!

---

## ⚠️ Важно — не сломайте формат!

- Не удаляйте **кавычки** `"` — текст всегда должен быть внутри них
- Не удаляйте **запятые** `,` между строками
- Не удаляйте **фигурные скобки** `{ }` и **квадратные скобки** `[ ]`
- Если хотите оставить поле пустым — оставьте `""`

## Поля сотрудника

| Поле | Что писать | Пример |
|------|-----------|--------|
| `name` | Имя (и Отчество, если нужно) | `"Наталья"` |
| `role` | Должность | `"Воспитатель младшей группы"` |
| `emoji` | Иконка (отображается, если нет фото) | `"👩‍🏫"` |
| `photo` | Имя файла фото из `public/images/team/` | `"natalya.jpg"` |
| `birthday` | День рождения | `"15 июня"` |
| `education` | Образование / специальность | `"Высшее педагогическое"` |
| `motto` | Жизненный девиз | `"Любить и учить!"` |
